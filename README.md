# ✨ Overview

Imran OS is a futuristic portfolio designed to replicate the experience of using a real desktop operating system.

Instead of scrolling through sections, users interact with applications, draggable windows, project folders, widgets, and desktop icons.

The project combines modern frontend technologies with glassmorphism and responsive UI principles to create an engaging developer portfolio.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/abdul-kadar-imran/Imran-Desktop.git
```

Navigate to the project directory

```bash
cd Imran-Desktop
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
